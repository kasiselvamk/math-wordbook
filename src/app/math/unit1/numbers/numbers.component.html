Natural Numbers And Whole Numbers

  1. Natural Numbers :

    Natural numbers are denoted by 'N'.numbers starting from '1 ' (one). N = {{ "{1, 2, 3, 4, 5, .........}" }}

  2. Whole Numbers :

    Whole numbers are denoted by 'W'.numbers starting from '0' (zero). W = {{ "{0, 1, 2, 3, 4, 5, .........}" }}

  3. Even Numbers :

    Numbers which are the multiples of '2' are called 'Even Numbers '. E = {{ "{2, 4, 6, 8, .........}" }}

  4. Odd Numbers :

    Numbers which are not the multiples of '2' are called 'Odd Numbers'. Odd = {{ "{1, 3, 5, 7, .........}" }}

  5. The smallest Natural Number is '1 ' and the smallest Whole Number is '0'.

  6. There do not exist the largest Natural number and the largest Whole number.

  7. The successor of any whole number is one more than that whole number.

    Ex : Successor of 11 is 12, (11 + 1)

  8. The predecessor of any whole number is one less than that whole number.

    Ex : Predecessor of 11 is 10 (11 - 1)

  9. There is no predecessor of zero in whole numbers.

  10. Face value and place value :

    • Face value of a digit is the digit itsell Ex.: Face value of '5' in 325 is '5'
    • Place value of a digit is the value of the digit in its place. Ex. : Place value of '5' in 352 is 50 [5 \times 10]

  11. Consecutive Numbers :

    The Numbers which differ from each other by one are called 'Consecutive Numbers'. Ex. : 13, 14 and 19, 20 are Consecutive Numbers

Properties Of Natural Numbers and Whole Numbers

    Let a, b and c be any three Whole Numbers. Then

  1. Closure Property :

    • a + b is a whole number Ex: 3 + 4 = 7 is a whole number
    • a \times b is a whole number Ex: 3 x 4 = 12 is a whole number

  2. Commutative Property :

    • a + b = b + a Ex: 4 + 5 = 5 + 4 = 9
    • a \times b = b \times a Ex: 4 x 5 = 5 x 4 = 20

  3. Associative Property :

    • a + (b + c) = (a + b) + c Ex: 3 + (4 + 5) = (3 + 4) + 5 = 12
    • a \times (b \times c) = (a \times b) \times c Ex: 3 x (4 x 5) = (3 x 4) x 5 = 60

  4. Distributive Property :

    • a \times (b + c) = (a \times b) + (a \times c) Ex: 3 x (4 + 5) = (3 x 4) + (3 x 5) = 27
    • a \times (b - c) = (a \times b) - (a \times c) Ex: 3 x (5 - 4) = (3 x 5) - (3 x 4) = 3
    • a + 0 = 0 + a = a
    • a - 0 = a
    • a \times 1 = 1 \times a = a
    • a + 1 = a
    • a - b \neq b - a
    • a \times 0 = 0 \times a = 0
    • 0 + a = 0 and {{"\\frac{a}{0}"}} is not defined.
