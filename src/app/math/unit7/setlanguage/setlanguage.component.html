    a well defined collection of objects

    A set which has a finite number of elements is called a "finite set."

    Ex : x = (a, e, i, o, u)

  6. Null Set (or) Empty set :

    A set containing no elements is called a null set (or) empty set (or) void set and it is denoted by 0 (or) ().

    Ex : Set of whole numbers greater than 8 but less than 5.

  7. Infinite Set :

    A set which has an infinite number of elements is called an infinite set.

  8. Equal Sets

    Two sets A and B are called equal sets if both contain the same elements.

  9. Equivalent Sets

    Two sets A and B are called equivalent sets if they contain the same number of elements.

  10. All equal sets are equivalent but all equivalent sets need not be equal.

  16. Union of Sets :

    The union of two sets 'A' and 'B' is A U B. It denotes all elements of the two sets.

    AUB = (x : x e A (or) x e B (or) x e both A and B)

    Note : The common elements 'a' and 'e' have to be taken only once while writing A U B.
