QUADRATIC EQUATIONS

  1. Quadratic Polynomial :

    A polynomial of second degree is caled a quadratic polynomial.

    Eg. x^2 - 3

    4x^2 - 5x + 6.

  2. Zeros of a Quadratic polynomial :

    The value of x in a polynomial for which polynomial function becomes zero, is called zero(s) of the given polynomial.

    If 'a' is a zero of the polynomial f(x)\;then\;f(a) = 0.

  3. Quadratic Equation :

    An equation of degree 2 is a quadratic equation.

    Eg : 2x^2 -4x+ 7 =0

    ax^2 + bx + c = 0

  4. Nature of the Roots of the equation ax^2 + bx+ c = 0

    Discriminent : D = b^2 -4ac

    (i) If D>O - unequal and real roots

    (ii) If D-O - equal and real roots

    (iii) If D < O - No real roots.
